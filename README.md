Jaywapp.Wpf
===========

WPF converters and helpers used across Jaywapp projects.

Targets
- net48
- net6.0-windows7.0
- net8.0-windows7.0

Install
PM> Install-Package Jaywapp.Wpf

Notes
- Requires Windows TFMs (WPF).
- XML documentation is included.
